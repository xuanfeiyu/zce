# 选择题

1、下面关于 asyncData 中描述错误的是

A. asyncData 中没有 this

B. 只有路由页面组件可以使用 asyncData 方法

C. asyncData 方法可以返回 Promise

D. asyncData 只会在服务端渲染期间调用执行

答案： D
